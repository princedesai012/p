# **App Name**: Bharat RateWatch

## Core Features:

- Real-time Gold & Silver Rates: Fetch and display current gold rates (22K & 24K) and silver rates from reliable sources.
- Petrol & Diesel Prices (City-Wise): Retrieve and show petrol and diesel prices for various cities in India. Display in cards using data fetched from API. LLM tool determines relevant formatting from API documentation.
- LPG Gas Price Display: Fetch and display the latest LPG gas prices.
- Vegetable & Onion Prices: Provide daily prices for key vegetables and specifically onions.
- City Selector: Implement a dropdown menu for users to select their city for accurate petrol and diesel prices.
- Comparison Chart: Generate a chart comparing today's prices with yesterday's for gold, silver, petrol, and diesel.
- Automatic Daily Updates: Implement a mechanism to automatically update prices daily to ensure data freshness.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to convey trust and reliability in financial data.
- Background color: Light blue (#D1E9F5), a desaturated variant of the primary color.
- Accent color: Vibrant orange (#F07124) for call-to-action buttons and important highlights.
- Body and headline font: 'Inter', a sans-serif font for a clean, modern, readable style.
- Use flat, outline-style icons to represent each type of price data (gold, petrol, vegetables, etc.).
- Implement a card-based layout to display price information, ensuring a clean and organized presentation.
- Use subtle animations like fade-ins and smooth transitions when data updates or when switching between cities.
